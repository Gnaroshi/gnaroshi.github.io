# Media Approval

## Stage 1 Status

- Production Hero: unchanged.
- Production favicon/brand: unchanged.
- Production project media: unchanged.
- Production functional icons: unchanged.
- Rejected diagrams and unused covers: identified, not yet deleted.

## Required Owner Input

Provide:

```text
Hero: <candidate ID or reject all>
Brand: <candidate ID or keep current>
Icons: approve candidate language / request changes
Project captures: approve collection plan / request changes
```

An approval may include crop or material adjustments. Silence or a general positive comment is not an approved ID.

## Stage 2 Gate

Stage 2 may start only after explicit IDs are recorded below.

- Approved Hero: _none_
- Approved brand: _none_
- Approved icon language: _none_
- Approved project capture plan: _none_
- Reviewer/date: _pending_

After approval, Stage 2 creates production variants, captures real evidence, replaces icon usage, removes legacy assets/allowlists, runs complete visual QA, and commits with `refactor: apply approved media and icon system`.
